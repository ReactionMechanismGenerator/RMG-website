    Definition of Inputs

    Required inputs:

    • Solvent SMILES: SMILES of a solvent
    • Solute SMILES: SMILES of a solute
    • Temp: temperature in K
    Optional input set 1 - one can provide a reference solubility value of the same solute in a different solvent and/or at a different temperature to get a more accurate solubility prediction. All of the three values must be provided together for this option.
    • Ref. Solvent SMILES: SMILES of a reference solvent
    • Ref. Solubility: solubility of the input solute in a reference solvent at a reference temperature. In logS (log10(mol/L)) unit.
    • Temp: reference temperature in K
    Optional input 2 - one can provide any or all of the values below to improve the temperature-dependent solubility prediction. Typically, ΔHsub298 is more important than Cpg298 and Cps298 to get a more accurate temperature-dependent solubility prediction.
    • ΔHsub298: sublimation enthalpy of the input solute at 298 K in kcal/mol
    • Cpg298: Heat capacity of the input solute in a gas phase at 298 K in cal/mol/K
    • Cps298: Heat capacity of the input solute in a solid phase at 298 K in cal/mol/K
